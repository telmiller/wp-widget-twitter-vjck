== Description ==

Twitter Widget displays your tweets on your sidebar. There's lots of switches to tuen on/off what you want to display.

== Installation ==

Place the all  files in your /wp-content/plugins/twitter directory
and activate through the administration panel, and then go to the widget panel and drag it to where you would like to have it.

IMPORTANT:
you need to register 'your' website onto twitter application by using following web site. to do so,

1) open https://dev.twitter.com/apps
2) sign in with your account
3) click 'create new application' button
4) follow the instruction and save
5) copy and paste:
- Consumer key
- Consumer secret
- Access token
- Access token secret
onto wedget's control panel.
6) save

*you need PHP version 5.3.0 or higher.
